## Como Usar (para Luna)

1. **Antes de criar qualquer criativo**, consultar este INDEX
2. **Buscar por procedimento** ou por **elemento de design** (tipografia, layout, etc.)
3. **Mesclar elementos** de diferentes referências (posição de foto de um + tipografia de outro)
4. **Nunca inventar estilo do zero** — sempre baseado em referências validadas

## Regras de Consulta (para Luna)

1. **Procedimento estético** → consultar seção por procedimento primeiro
2. **Estilo de criativo** → consultar coleções Behance
3. **Elemento específico** (tipografia, callout, etc.) → consultar tabelas de elementos
4. **Mesclagem** → sempre combinar 2-3 referências, NUNCA copiar 1 referência inteira
5. **Validação** → após criar, comparar com referências similares do swipe file
