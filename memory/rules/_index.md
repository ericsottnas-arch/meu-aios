## PASSO 1: Sempre ler primeiro

[[universal]] — Regras que se aplicam a TODA e QUALQUER tarefa, sem excecao.

---

## PASSO 2: Identificar o tipo de tarefa e ler o arquivo certo

### Produzindo qualquer texto (copy, roteiro, email, caption, headline)?
→ Ler: [[copy-escrita]]
→ Se cliente medico: tambem ler [[clientes-medicos]]

### Construindo ou editando landing page / HTML?
→ Ler: [[landing-pages]]
→ Se cliente medico: tambem ler [[clientes-medicos]]

### Gerindo campanhas de trafego pago (Meta Ads, Google Ads)?
→ Ler: [[ads-campanhas]]

### Produzindo conteudo para redes sociais (@nova)?
→ Ler: [[conteudo-redes-sociais]]

### Trabalhando com clientes medicos, estetica, HOF, dermatologia?
→ Ler: [[clientes-medicos]]

### Ativando agente / executando qualquer workflow de agente?
→ Ler: [[agentes-comportamento]]

### Criando documento, relatorio, briefing para cliente?
→ Ler: [[entrega-documentos]]
→ Tambem ler: [[copy-escrita]] (regras de tom)

### Configurando automacoes, webhooks, GHL, codigo?
→ Ler: [[dev-automacoes]]

## Como usar cross-referencias

Cada arquivo de regras pode referenciar outros com [[nome-do-arquivo]].
Ao encontrar um link, LEIA o arquivo referenciado antes de continuar.
