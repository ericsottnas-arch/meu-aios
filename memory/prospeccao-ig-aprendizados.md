## PADRÕES IDENTIFICADOS NESSA SESSÃO

### O que está funcionando
- Comentário técnico específico sobre o procedimento mostrado → abre conversa naturalmente
- Quando o prospect usa automação/bot → entra por aí, não pelo marketing
- Perguntar sobre o que ELA valoriza (metodologia, processo) → ela fala, você aprende, vínculo cresce

### Ferramenta /audio + /screenshot
- Workflow: screenshot para ver o perfil/story → /audio para ouvir o vídeo → transcrição → mensagem personalizada
- Funciona: transcrição captura detalhes específicos que a mensagem pode referenciar
- Exige Multi-Output Device ativo como saída do sistema (configurado 17/03/2026)

### Tipo de conteúdo — quando NÃO abordar
- Story de panfleto (card de serviços, lista de procedimentos, endereço/contato) → PULAR
- Comentar nesses stories soa igualmente comercial e não gera conexão real
- Esperar conteúdo pessoal: resultado, bastidor, opinião, processo de atendimento

### Regras de tom confirmadas (desta sessão)
- Sem aspas nos scripts
- Sem traço/dash
- Sem formalidade ("Dra. X", "tenho acompanhado")
- Curto, casual, como texto de celular
