### Before You Start
1. Ensure you have appropriate permissions for data collection
2. Review Instagram's Terms of Service
3. Comply with LGPD (Brazil's data protection law)
4. Prepare disclosure statements for any affiliate/sponsored content

### Data Privacy Compliance
- [ ] Get consent before featuring customer data
- [ ] Anonymize sensitive information
- [ ] Include proper disclaimers
- [ ] Handle before/after photos responsibly
- [ ] Protect customer privacy

### Instagram Policy Compliance
- [ ] Don't automate logins or direct messaging
- [ ] Use official APIs where available
- [ ] Respect rate limits and quotas
- [ ] Disclose sponsored content
- [ ] Include proper hashtags and descriptions
