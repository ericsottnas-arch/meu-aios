## FRAMEWORKS DE COPY (Quick Reference)

### PASTOR (Ray Edwards)
P = Person, Problem, Pain → A = Amplify → S = Story, Solution → T = Transformation → O = Offer → R = Response

### Rule of One (Masterson)
1 Ideia + 1 Emoção + 1 História + 1 Benefício + 1 CTA

### 4 U's Headlines (Masterson)
Useful + Urgent + Unique + Ultra-Specific (nota 1-4 em cada)

### CUB Critique (Masterson)
C = Confusing? U = Unbelievable? B = Boring? → Marcar e corrigir

### Four-Legged Stool (Masterson)
1. Big Idea Emocional | 2. Promessa de Benefícios | 3. Prova | 4. Credibilidade

### Slippery Slide (Sugarman)
Hook → Seeds of Curiosity → Open Loops → "Mas tem mais..." → CTA inevitável

### 6 Lead Types (Masterson/Great Leads)
1. Offer Lead (Most Aware) → 2. Promise → 3. Problem-Solution → 4. Big Secret → 5. Proclamation → 6. Story Lead (Unaware)

### Attractive Character (Brunson)
Backstory + Parables + Falhas + Polaridade → 4 arquétipos: Líder, Aventureiro, Repórter, Herói Relutante

### 6 $tory Types (Garfinkel)
1. Origin → 2. Prospect's Pain → 3. Future Pacing → 4. Reassurance → 5. Explanation → 6. Trust Building

## REGRAS DOS PERFIS DE REFERÊNCIA

### Posicionamento (de @iallas)
- "Não basta ser bom. É preciso ser VISTO como bom"
- Positioning > Posting — aparecer DO JEITO CERTO, não mais vezes
- Quem se posiciona bem não disputa preço, agenda ou atenção

### Conteúdo como Vendas (de Alfredo Soares)
- "Ser mais interessante do que interesseiro" — eduque ANTES de vender
- Todo profissional É uma marca — construa ecossistema
- "A melhor estratégia é atitude" — comece hoje

### Volume e Dominação (de Grant Cardone)
- 10X o volume de conteúdo — a maioria posta pouco demais
- Domine, não compita — seja A referência
- Preço nunca é a objeção real — é falta de valor percebido
- "Be sold first" — se o profissional não acredita, o paciente nunca vai

### Processo de Vendas (de Full Sales System)
- Modelo Setter-Closer para clínicas: recepção qualifica, consultor fecha
- Social Selling: conteúdo orgânico É funil de vendas, não branding
- Referral = lead quente — tratar com máxima urgência
- Testar orgânico, escalar com ads

### Lifestyle (de @eusouroberth)
- "Furar a bolha" — romper limitações invisíveis
- Elevação holística: pessoal + emocional + profissional juntos
- "Alto Padrão" como filosofia de vida e posicionamento

## 7 PRINCÍPIOS DE CIALDINI (Quick Reference)

1. **Reciprocidade** — Dê valor primeiro, a obrigação de retribuir compensa
2. **Compromisso** — Micro-commits levam a ações maiores
3. **Prova Social** — Números + similaridade + proximidade
4. **Autoridade** — Títulos + credenciais + jaleco + publicações
5. **Liking** — Bastidores + similaridade + elogios + presença constante
6. **Escassez** — Vagas limitadas + deadline + loss framing (NUNCA falsa)
7. **Unidade** — Identidade tribal: "nós" > "eu e você"
