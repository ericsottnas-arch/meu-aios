## DNA DE COPY (extraído das referências)

**Regras absolutas:**
- Usa "tu/teu/você" — direto, sem distância
- Reframe brutal: o que a pessoa acha que é força é na verdade o problema
- Dados concretos, nunca vago ("87%", "R$5.500", "10 dias")
- Urgência de mercado ("Em 2026...", "o mercado não perdoa...")
- Frases curtas. Uma ideia por parágrafo
- Nunca começa com pergunta (afirmação é mais forte)
- Termina com gancho pro próximo slide ou CTA direto

**O tapa de realidade (principal mecanismo):**
> Você pega algo que a pessoa acha que é mérito ("minha clínica tem boa reputação")
> e mostra que é exatamente o que está te impedindo de crescer ("reputação não paga conta, paciente novo paga")

### FORMATO 2 — "Carrossel Frase Minimalista" (Dados)
**Fonte:** @ohenriquecedor
**Visual:** Fundo branco · texto bold centralizado · uma informação por slide
**Quando usar:** Educar com dados, criar urgência quantificada

**Estrutura (5–7 slides):**
```
Slide 1: dado chocante que para o scroll
Slide 2: dado que amplifica
Slide 3: dado que aprofunda
Slide 4: dado que causa dor
Slide 5: pergunta que confronta (sem resposta, deixa o lead pensar)
```

**Exemplo para Eric:**
```
[Slide 1]
80% dos pacientes fecham
entre o 5º e o 10º contato

[Slide 2]
Mas 70% das clínicas
desistem no 1º

[Slide 3]
O lead não sumiu.
Você parou de aparecer.

[Slide 4]
Cada lead sem follow-up =
R$300–R$2.000 jogado fora

[Slide 5]
Quantos leads você perdeu
nos últimos 30 dias?
@byericsantos
```

**Frequência sugerida:** 1x/semana · alta retenção, salvo frequentemente

### FORMATO 3 — "Thread Preta" (Carrossel opinionado)
**Fonte:** @alfredosoares
**Visual:** Fundo preto · profile pic Eric + @byericsantos no topo · parágrafos curtos
**Quando usar:** Take forte, opinião de mercado, reframe de crença

**Estrutura:**
```

[Último slide] CTA direto: "Salva esse post" / "DM pra saber mais"
```

**Exemplo para Eric:**

```
Você não tem problema de captação.
Você tem problema de sistema.

   O lead foi embora.

[Slide 3]
2. O lead respondeu, fez orçamento,

   O lead foi pro concorrente.

[Slide 4]
3. O lead fechou um procedimento.
   Ninguém ofereceu o próximo.
   Você perdeu a recorrência.

[Slide 5]
Isso não é azar.
É ausência de processo.

E processo se constrói em 30 dias.
```

**Frequência sugerida:** 1x/semana · maior engajamento, salvo e compartilhado

### FORMATO 4 — "Estático Tweet" (Post + análise)
**Fonte:** @alfredosoares
**Visual:** Imagem que parece tweet/post embutido (fundo escuro) + análise na legenda
**Quando usar:** Comentar cases reais, notícias de mercado, provocação de alto nível

**Estrutura:**
```
[Imagem] Screenshot de tweet/post ou pergunta bold em fundo escuro
[Legenda] Análise completa: 3 pontos de insight + aplicação para o ICP
```

**Exemplo para Eric:**
```
[Imagem]
"Clínica com 500 seguidores fecha mais
que clínica com 50.000?"

Minha análise na legenda.

[Legenda]
Sim. E tem 3 motivos:

1. Autoridade não é número de seguidor.
   É clareza de quem você atende e qual problema resolve.

2. 500 seguidores certos valem mais que 50k aleatórios.
   Um feed de médico estético falando com dono de clínica
   converte diferente de um feed falando pra todo mundo.

3. O algoritmo não vende. Você vende.
   Instagram é vitrine. Processo é o que fecha.

Você está construindo vitrine ou sistema?
```

**Frequência sugerida:** 1x/semana · posiciona como analista de mercado

### FORMATO 5 — "Reel com Narrativa" (já no seu feed)
**Fonte:** Seus posts atuais + vídeo de referência salvo
**Visual:** Talking head, texto overlay bold
**Quando usar:** Bastidores, casos reais, provocação em vídeo

**Narrativa do vídeo de referência salvo:**
Follow-up conversacional com lead que orçou mas não converteu.
Tom: empático, sem pressão, curioso, resolve objeção antes de vender.
Frase-chave: *"Não é pra te comprar nada. Quero entender o que te impediu."*

**Adaptação para Eric:**
```
Hook (0–3s): Afirmação que para o scroll
Body (3–20s): Desenvolvimento do problema real
Reframe (20–25s): O que muda com sistema
CTA (25–30s): Um comando simples
```

**Frequência sugerida:** 3–4x/semana (já é seu formato principal)
