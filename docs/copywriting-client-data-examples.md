
## 🎯 Checklist: Como Copywriter Deve Usar Dados

### Antes de Escrever:

- [ ] `*client-{seu-comando}` para acessar dados específicos
- [ ] Revisar headlines vencedoras históricas
- [ ] Entender pain/gain points do público
- [ ] Verificar performance anterior (baseline a superar)
- [ ] Confirmar brand voice esperada
- [ ] Identificar gatilhos psicológicos que funcionam

### Durante a Escrita:

- [ ] Usar padrões que já funcionaram como baseline
- [ ] Incorporar social proof documentada
- [ ] Aplicar frameworks que tiveram sucesso
- [ ] Testar variações dos headlines vencedores
- [ ] Manter brand voice consistente

### Depois de Lançar:

- [ ] Documentar performance vs baseline
- [ ] Registrar o que funcionou (headlines, frameworks, estruturas)
- [ ] Anotar objeções que surgiram
- [ ] Preparar learnings para próxima campanha
