# Copywriting Hierarchy - Quick Start Guide

## TL;DR: Sua Hierarquia de Copywriting

```
🎭 COPY-CHEF (Orquestrador Principal)
   └─ Identifica demanda → Roteia para especialista certo → Revisa qualidade

📊 6 ESPECIALISTAS LENDÁRIOS (Clones de Maiores Nomes do Copywriting)
   ├─ @halbert        ✉️  Direct Response & Sales Letters (Gary Halbert)
   ├─ @ogilvy         🎨  Brand Copywriting & Advertising (David Ogilvy)
   ├─ @wiebe          📱  Conversion & Landing Pages (Joanna Wiebe)
   ├─ @georgi         💎  High-Ticket Sales & Consultoria (Stefan Georgi)
   ├─ @orzechowski    📧  Email Marketing & Sequences (Chris Orzechowski)
   └─ @morgan         👩  Female-Focused Copywriting (Lorrie Morgan)
```

---

## 3 Passos Para Começar

### 1️⃣ Ativar o Copy-Chef
```bash
@copy-chef
```
Copy-Chef é seu ponto de entrada. Ele identifica que tipo de cópia você precisa.

### 2️⃣ Descrever a Demanda
```
"Preciso de copy para landing page de um programa $497/mês para mulheres empreendedoras"
```
Quanto mais detalhes, melhor. Copy-Chef faz perguntas se precisar.

### 3️⃣ Copy-Chef Roteia Para Especialista
Copy-Chef vai analisar e rotear:
- Tipo de cópia (email, sales page, landing page, etc)
- Objetivo (conversão, leads, vendas, awareness)
- Público (mulheres, empreendedores, B2B, etc)
- Tamanho da oferta ($47 vs $5K)

## Os 6 Especialistas em 1 Minuto

### ✉️ @halbert - Gary Halbert (Direct Response Master)
- **Especialidade:** Sales letters, headlines que MATAM, emails frios
- **Foco:** Resposta imediata, conversão máxima, testing obsessivo
- **Filosofia:** "Cada palavra justifica sua existência ou é deletada"
- **Quando usar:** Precisa de máxima conversão/resposta rápida

### 🎨 @ogilvy - David Ogilvy (Brand Storyteller)
- **Especialidade:** Brand positioning, advertising sofisticado, premium messaging
- **Foco:** Brand equity longo prazo, storytelling elegante
- **Filosofia:** "Copy é apenas tão boa quanto a ideia por trás"
- **Quando usar:** Quer posicionar marca premium, brand awareness

### 📱 @wiebe - Joanna Wiebe (Conversion Scientist)
- **Especialidade:** Landing pages, form optimization, CRO
- **Foco:** Taxa de conversão máxima, A/B testing, psychology triggers
- **Filosofia:** "Teste uma variável por vez, sempre baseado em dados"
- **Quando usar:** Quer otimizar taxa de conversão de página

### 💎 @georgi - Stefan Georgi (High-Ticket Sales Master)
- **Especialidade:** Ofertas premium ($5K+), consultoria, sales pages sofisticadas
- **Foco:** Qualificação brutal, disqualificação estratégica, objection handling
- **Filosofia:** "Não é volume, é VALOR. Quer os clientes certos"
- **Quando usar:** Vende oferta premium, precisa de email sequence estratégica

### 📧 @orzechowski - Chris Orzechowski (Email Maestro)
- **Especialidade:** Email sequences, automation, nurture funnels
- **Foco:** Engajamento, retenção, subject lines vencedoras
- **Filosofia:** "Email é o canal com maior ROI"
- **Quando usar:** Precisa de email sequence automática ou campainha

### 👩 @morgan - Lorrie Morgan (Female Psychology Expert)
- **Especialidade:** Copy para mulheres, community building, emotional connection
- **Foco:** Psicologia feminina, transformação, pertencimento
- **Filosofia:** "Mulheres compram em emoção + lógica + relacionamento"
- **Quando usar:** Público-alvo é mulheres ou precisa de connection emocional

## Fluxo Completo (Passo a Passo)

```
VOCÊ:
"Preciso de landing page para programa $297 de email marketing"

↓

COPY-CHEF:
Faz perguntas e analisa:
✓ Tipo: Landing page
✓ Objetivo: Conversão
✓ Público: Empreendedores
✓ Offer: $297

↓

ROTEIA PARA: @wiebe
Razão: Landing page = conversão = Wiebe especialista

↓

@WIEBE:
Recebe briefing e escreve:
✓ Headlines vencedoras (3+ variações para test)
✓ Copy focado em conversão
✓ Form otimizado
✓ CTA claro e repeated
✓ A/B test plan

↓

COPY-CHEF REVISA:
✓ Framework foi aplicado?
✓ Persuasão está máxima?
✓ CTA está claro?
✓ Brand voice consistente?

↓

VOCÊ RECEBE:
Landing page copy pronta para lançar

↓

FEEDBACK LOOP:
Você testa → Coleta performance → Share com Copy-Chef
→ Copy-Chef otimiza roteamento futuro
```

## Key Principles

### ✅ DO
- Use Copy-Chef como entry point
- Forneça briefing claro
- Rastreie performance
- Documente feedback
- Teste uma variável por vez

### ❌ DON'T
- Não pule Copy-Chef intake
- Não ignore review de qualidade
- Não tome decisões com <72h dados
- Não mude múltiplas variáveis ao mesmo time
- Não escreva copy sem briefing
