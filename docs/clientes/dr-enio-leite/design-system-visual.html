07 / Regras
O que fazer.
O que nunca fazer.

Diretrizes para manter a identidade consistente.

✓   F A Z E R
Playfair Display para headlines

Serif editorial com contraste elegante. É a voz do Instituto.

Italic como overlay de destaque

Playfair Italic em dourado sobreposto ao headline bold. Assinatura visual.

Fundo creme como base

#F7F4EF. Escuro só em hero e CTA para ancorar a hierarquia.

Preto para peso visual

#15120D em headlines — contraste máximo contra fundo claro.

Spaced caps nos labels

DM Sans 200, 0.28em spacing, uppercase. Sofisticação instantânea.

✗   N U N C A
Fundo escuro dominando

Escuro só em seções de âncora. Não como fundo geral.

Gold em grandes blocos

Bronze é acento cirúrgico — italic overlays e labels. Nunca fundo.

Terceira fonte

Playfair + DM Sans. Duas famílias é o máximo.

Border-radius em cards

Sempre 0. Bordas arredondadas = casualidade.

DM Sans bold em display

Headlines grandes = sempre Playfair. DM Sans fica pesado.
