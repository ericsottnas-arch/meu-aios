# Copywriting Patterns - Dra. Gabriele Estética

**Objetivo:** Servir como referência para especialistas de copy reproduzirem **tom, estrutura, mecanismos emocional/racional** da Dra. Gabriele

**Fonte:** Análise dos 3 roteiros | Padrões linguísticos + psicológicos

---

## 🎯 ARQUITETURA DE COPY (Estrutura Padrão)

Cada roteiro segue esta estrutura:

```
[CABEÇAS MÚLTIPLAS: Pain points + Objections]
     ↓
[💙 CORPO ÚNICO: Validação + Mecanismo + Diferencial]
     ↓
[💙 CORPO ÚNICO: Antes/depois prova]
     ↓
[CTA 1-3: Soft → Warm → Hard]
```

### Detalhamento

#### CABEÇAS (2-5 per produto)
- **Formato:** 🔴 CABEÇA X - [Headline emoção]
- **Função:** Evocar dor específica, criar identificação
- **Linguagem:** Conversacional, narrativa visual
- **Exemplos:**
  - "Você tem aquela calça jeans guardada no armário..."
  - "Você usa aquele shortinho por cima da legging..."
  - "Você compra biquíni e a primeira coisa que você pensa é..."

**Pattern Observado:** Cada cabeça é uma cena visual/emocional que cliente reconhece EM SUA VIDA

#### CORPO ÚNICO (💙)
- **Formato:** Parágrafo conversacional + proposição clara
- **Função:** Validar, educar, posicionar diferencial
- **Linguagem:** "Olha, eu vou te falar...", "A verdade é...", "Não é culpa sua..."
- **Estrutura interna:**
  1. Admitir verdade (o que clínicas rápidas não fazem)
  2. Validar frustração (não é culpa)
  3. Explicar mecanismo (por quê não funciona)
  4. Introduzir método (eu criei...)
  5. Oferecer prova (antes/depois)

**Pattern Observado:** Sempre "EU" (executora de resultado), nunca "clínica oferece"

#### CTA HIERARCHY
- **CTA 1 - Suave:** Follow Instagram (lowest friction)
- **CTA 2 - Conversacional:** WhatsApp (warm conversation)
- **CTA 3 - Urgente:** Formulário (hard commitment + escassez)

## 🗣️ FRASES ABERTURA (Gatilhos de Atenção)

Padrões que iniciam cada seção:

### "Olha, eu vou..." (Sinceridade + Confessoional)
```
"Olha, eu vou ser bem sincera com você: o seu corpo é muito mais complexo..."
"Olha, eu vou te falar uma coisa que ninguém comenta..."
"Olha, eu vou te falar a verdade que ninguém te conta..."
```
**Efeito:** Cria pacto de autenticidade, diferencia de "marketing speak"

### "Você tem..." (Identificação Visual)
```
"Você tem aquela calça jeans guardada no armário..."
"Você treina 4, 5 vezes por semana, segue a dieta direitinho, mas..."
"Você precisa usar cinto em TODAS as calças jeans..."
```
**Efeito:** Colocas cliente DENTRO da narrativa, reconhecimento imediato

### "Quantas vezes..." (Padrão Repetitivo)
```
"Quantas vezes você já começou uma dieta segunda-feira..."
"Quantas vezes você já somou quanto dinheiro você gastou..."
```
**Efeito:** Agrega frustração (1x bad, 3x+ = padrão crônico)

### "Existe..." (Revelação)
```
"Existe um procedimento que todo mundo faz mas NINGUÉM assume..."
```
**Efeito:** Mistério + validação (você não é só você pensando)

## ❤️ VALIDAÇÃO (O Tom Maternal-Executivo)

### Frases de Validação (Não é Culpa Sua)
```
"Não é culpa sua ter essa rotina maluca..."
"Não é culpa sua o treino não funcionar..."
"Não é culpa sua, é genética mesmo..."
"Não é culpa sua, é [inflamação/hormônio/cortisol alto]..."
```

**Padrão:** [Situação] → [Não é culpa] → [Razão biológica/real]

**Efeito Emocional:** Transição de autocrítica → autoaceitação → solucionismo

### Frases de Empatia (Eu Entendo)
```
"Eu vi muita mulher frustrada chegando aqui depois de já ter tentado de tudo..."
"Eu desenvolvi um método justamente pra mulher que TÁ na correria..."
"Eu sou que fiz, cliente que tinha desistido de tudo..."
```

**Padrão:** [Observação] → [Empatia] → [Ação de mim]

**Efeito:** Posiciona como expert + companheira (não apenas prestadora de serviço)

## 🔬 EDUCAÇÃO (Mecanismo Científico Simplificado)

### Pattern: [Problema] → [Causa Biológica] → [Solução Sistemática]

#### Exemplo Lipo Sem Cortes:
```
PROBLEMA: "A gordura simplesmente não quebra"
CAUSA: "Se você tá inchada e inflamada..."
SOLUÇÃO: "A gente desInflama o seu corpo primeiro"
```

#### Exemplo Power Shape:
```
PROBLEMA: "Essa gordurinha que fica ali na pochete não sai só com treino..."
CAUSA: "...é gordura RESISTENTE, seu corpo segura ela ali por causa de hormônio, estresse, cortisol alto"
SOLUÇÃO: "O Power Shape trabalha JUNTO com seu treino pra quebrar essa gordura teimosa"
```

#### Exemplo Glúteo:
```
PROBLEMA: "Se você não tem genética pra bumbum grande, treino sozinho NÃO vai fazer ele crescer"
CAUSA: "...se o seu corpo não foi programado pra acumular volume ali..."
SOLUÇÃO: "...por isso eu comecei a fazer preenchimento de glúteos..."
```

**Padrão Observado:** Nunca culpa cliente. Sempre nomeia raiz verdadeira (genética, hormônio, inflamação)

## 🎁 DIFERENCIAL (Proprietário + Prova)

### "Eu Criei..." (Propriedade + Inovação)
```
"Eu desenvolvi um método aqui na clínica chamado Lipo Sem Corte..."
"Eu desenvolvi um método aqui na clínica chamado Power Shape..."
"Eu comecei a fazer preenchimento de glúteos..."
```

**Padrão:** [EU] + [CRIEI/DESENVOLVI] + [MÉTODO PRÓPRIO] + [PORQUE VI FRUSTRAÇÃO]

**Efeito:** Posiciona como inventor, não replicadora. Credibilidade de obsessão

### "Tá Vendo..." (Proof via Visual)
```
"Tá vendo esse antes e depois aqui? Eu que fiz, cliente que tinha desistido de tudo e achava que nunca ia conseguir."
"Tá vendo esse resultado aqui na tela? Essa cliente sofria EXATAMENTE com isso..."
"Tá vendo essa transformação aqui? Ela tinha feito em 3 lugares diferentes e nada funcionava..."
```

**Padrão:** [VISUAL PROOF] + [EU QUE FIZ] + [CONTEXTO CLIENTE]

**Efeito:** Invoca visualização + credibilidade pessoal + empatia com cliente

### "Não é Só..." (Scarcity + Exclusivity)
```
"Não é só eliminar a pochete, é moldar o corpo que você tanto treina mas que não aparece..."
```

**Padrão:** [Vai além de expectativa]

**Efeito:** Premium positioning (não é transação básica, é transformação)

## 🎪 ANTES/DEPOIS NARRATIVA (Social Proof Structure)

### Pattern Padrão:

**ANTES:**
- [Situação específica cliente]
- [Esforço que fez]
- [Resultado que NÃO veio]
- [Emoção = frustração]

**DEPOIS:**
- [Resultado visível]
- [Transformação de vida]
- [Prova = antes/depois real, sem filtro]

### Exemplos Estruturados:

**Lipo Sem Cortes:**
```
ANTES: "Cliente que tinha desistido de tudo"
ESFORÇO: [implícito - tentou tudo]
FALHA: "Tinha desistido de tudo e achava que nunca ia conseguir"
EMOÇÃO: Impotência

DEPOIS: "Tá vendo esse antes e depois aqui?"
TRANSFORMAÇÃO: [Visual]
PROVA: "EU que fiz"
```

**Power Shape:**
```
ANTES: "Essa cliente sofria EXATAMENTE com isso, treinava pesado, comia bem, mas a pochete não saía de jeito nenhum"
EMOÇÃO: Frustração extrema (fez tudo certo, ainda falhou)

DEPOIS: "Hoje olha o resultado que ela tem"
PROVA: "EU que entreguei"
```

**Glúteo:**
```
ANTES: "Essa cliente treinava há anos e o bumbum não crescia, ela tinha vergonha de usar shorts, biquíni"
EMOÇÃO: Vergonha + Limitação social

DEPOIS: "Hoje olha o resultado natural que ela tem"
BONUS: "Todo mundo pergunta pra ela qual exercício ela faz, ela só sorri porque sabe que foi o preenchimento que EU fiz"
```

**Padrão:** Antes/depois sempre inclui "EU QUE FIZ" (não médico, assistente, clínica - DRA. GABRIELE)

## 🎬 CTA COPYWRITING (3 Níveis)

### CTA 1 - SUAVE (Awareness → Interest)

**Estrutura:**
```
"Me segue aqui que [promise: o que você vai ver] [diferencial: real/sem filtro]"
```

**Exemplos:**
```
"Me segue aqui que todo dia eu mostro transformação real de mulher real, sem filtro, sem mentira."
"Me segue aqui que eu posto resultado real de mulher que treina e conseguiu se livrar da pochete de vez."
"Me segue aqui que eu mostro os resultados reais, sem filtro, de mulheres que fizeram e transformaram a autoestima."
```

**Padrão Observado:**
- Promessa + Prova (real, sem artifício)
- Call to follow (lowest friction)
- Frequência ("todo dia", "todo", "todos")
- Diferencial: "real", "sem filtro", "sem mentira"

**Efeito:** Cria pipeline de educação + community

---

### CTA 2 - CONVERSACIONAL (Interest → Consideration)

**Estrutura:**
```
"Clica aqui [e me manda mensagem / e me chama] no WhatsApp que eu vou te [explain/show] [benefit for your case]"
```

**Exemplos:**
```
"Clica aqui embaixo e me manda mensagem no WhatsApp que eu vou te explicar certinho como funciona e se faz sentido pro seu caso."
"Clica aqui e me chama no WhatsApp que eu te explico direitinho como o Power Shape funciona e se é pra você."
"Clica aqui e me chama no WhatsApp que eu te explico certinho como funciona, quanto custa e se é indicado pro seu caso."
```

**Padrão Observado:**
- Channel: WhatsApp (intimate, direct)
- Tone: "certinho", "direitinho" (diminutivo = carinho)
- Personalization: "se faz sentido pro seu caso", "se é pra você"
- Inclusivity: Pergunta, não impõe

**Efeito:** Qualificação suave + abre conversa 1-on-1

---

### CTA 3 - URGENTE (Consideration → Decision)

**Estrutura:**
```
"Preenche o formulário agora porque [scarcity reason: poucas vagas/clientes] e [consequence: não deixa acontecer novamente]"
```

**Exemplos:**
```
"Preenche o formulário agora porque eu só consigo atender um número pequeno de clientes por mês e as vagas enchem rápido, não deixa pra depois."
"Preenche o formulário agora porque eu só consigo atender algumas clientes por mês, não deixa essa pochete te incomodar por mais um ano inteiro."
"Preenche o formulário agora porque eu faço poucos procedimentos por mês e as vagas enchem rápido, não passa mais um verão escondendo o bumbum."
```

**Padrão Observado:**
- Scarcity: "poucos", "número pequeno", "vagas enchem rápido"
- Consequence: Descreve custo emocional de NÃO agir ("mais um ano inteiro", "passar verão escondendo")
- Ação: Preencher formulário (hard commitment)
- Urgência: "agora", "não deixa", "não passa"

**Efeito:** Cria Fear of Missing Out (FOMO) legitimamente justificada

## 💬 PADRÕES LINGUÍSTICOS

### Conversacionalidade (Anti-jargão)

❌ NÃO USA:
- "procedimento não-invasivo" (jargão)
- "tecnologia de ponta" (genérico)
- "clínica moderna" (vago)
- "maioria das mulheres" (impessoal)

✅ USA:
- "sem corte nenhum" (específico)
- "Eu que fiz" (pessoal)
- "você que treina" (direto)
- "gordura teimosa que tá ali" (visceral)

### Estrutura Frase
- **Frases curtas:** "O resultado? Não vem." (impacto)
- **Repetição:** "Não sai, simplesmente não vai" (ênfase)
- **Pergunta retórica:** "Você já somou quanto?"  (reflexão)
- **Diminutivos:** "Certinho", "direitinho" (carinho maternal)

### Emotividade
- **Frustração nomeada:** "bate aquela frustração"
- **Esperança oferecida:** "Tá vendo esse resultado que ela tem"
- **Raiva validada:** "Você pode fazer 500 agachamentos..."
- **Segurança oferecida:** "Não é culpa sua"

## 🎨 TONE OF VOICE GUIDELINES

### Por Contexto:

| Contexto | Tom | Exemplo |
|----------|-----|---------|
| **Validação** | Maternal + Sincera | "Não é culpa sua, é genética mesmo..." |
| **Educação** | Executiva + Clara | "A gordura simplesmente não quebra se..." |
| **Diferencial** | Confiante + Proprietária | "Eu desenvolvi um método aqui na clínica..." |
| **Proof** | Assertiva + Pessoal | "EU que fiz, cliente que tinha desistido..." |
| **CTA Soft** | Convite + Frequência | "Todo dia eu mostro transformação real..." |
| **CTA Hard** | Urgência + Consequência | "Não deixa essa pochete te incomodar..." |

### Nunca:

❌ Jargão medical/técnico sem explicação
❌ Posicionamento genérico ("melhor clínica", "inovador")
❌ Falsas promessas ("100% resultado")
❌ Jargão de coach ("transforme sua vida", "seja a melhor versão")
❌ Diminuição do problema ("pequeninha")

### Sempre:

✅ Sinceridade vulnerável
✅ Nomeação específica de frustração
✅ Mecanismo simples (genética, hormônio, inflamação)
✅ Proof pessoal ("EU que fiz")
✅ Validação emocional ("não é culpa sua")
