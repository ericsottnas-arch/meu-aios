## 🎯 Guia de Uso

### Para Encontrar Informações sobre um Cliente

1. **Público-Alvo & Personas:** Abra `cliente-nome/ICP.md` ⭐ (SOURCE OF TRUTH)
2. **Visão Geral Rápida:** Abra `cliente-nome/README.md`
3. **Briefing Completo:** Abra `cliente-nome/profile.md`
4. **Dados Estruturados:** Veja `cliente-nome/knowledge-base/data.json`
5. **Histórico:** Consulte `cliente-nome/history/`

### Para Gerenciar Clientes

- **Configurações Centralizadas:** `CLIENTES-CONFIG.json`
- **Análises Comparativas:** `ANALISE-COMPARATIVA.md`
- **Referências do Celo:** Ver cada `README.md`
- **ICPs de Todos os Clientes:** Ver cada `ICP.md`

### Para Adicionar Novo Cliente

1. Crie pasta: `docs/clientes/nome-cliente/`
2. Crie subpastas: `knowledge-base/`, `history/`, `assets/`
3. Adicione `README.md` e `profile.md`
4. **Crie ICP.md** baseado em `TEMPLATE-ICP.md` (NOVO!)
5. Atualize este INDEX

### Para Fazer Copy/Campaigns

1. **Leia ICP.md** do cliente (entenda as personas)
2. **Identifique qual persona** vai escrever
3. **Use messaging específico** dessa persona
4. **Use hook + story + CTA** específicos da persona
5. **Valide com checklist** do ICP
6. Consulte `GUIA-IMPLEMENTACAO-ICP.md` para mais detalhes
