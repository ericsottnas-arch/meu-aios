05 · Motion & Animação

Sistema de
movimento

suave como seda

Stack: GSAP 3.12.5 + ScrollTrigger + Lenis 1.1.18 (smooth scroll) + SplitType 0.3.4. Nunca animações abruptas. Cada movimento é intencional e elegante.

Curvas de Easing
--ease-out
cubic-bezier(.16,1,.3,1)
Entrada suave. Usado em reveals, slides, hovers
--ease-in
cubic-bezier(.7,0,.84,0)
Saída rápida. Usado em elementos que saem do viewport
--ease-inout
cubic-bezier(.87,0,.13,1)
Transições de estado: page transitions, modals
Durações
--dur-fast
200ms
Hovers, micro-interações
--dur-mid
400ms
Reveals, acordeões
--dur-slow
700ms
Entrada de seções, slides
--dur-xslow
1.2s
Loader, hero entrance
Stack de Animação
GSAP + ScrollTrigger
Scroll-driven animations, timelines complexas, parallax, pin sections. Padrão para animações de scroll.
v3.12.5 · CDN
Lenis Smooth Scroll
RAF-synced smooth scroll. Integrado com GSAP ScrollTrigger via lerp. Substitui scroll nativo.
v1.1.18 · CDN
SplitType
Text reveals por linha, palavra ou caractere. Sempre .revert() no cleanup para prevenir memory leaks em SPA.
v0.3.4 · CDN
Padrão de Reveal
gsap.from(".reveal", {
  opacity: 0,
  y: 36,
  duration: 0.9,
  ease: "power3.out",
  stagger: 0.08,
  scrollTrigger: {
    trigger: ".reveal",
    start: "top 82%"
  }
});
