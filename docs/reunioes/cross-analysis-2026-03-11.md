## 3. TOM DE VOZ DO ERIC — PADRÕES LITERAIS EXTRAÍDOS

### Estilo de comunicação
- Direto, consultivo, nunca vendedor
- Fala como colega de mercado, não como fornecedor
- Usa "a gente" (inclusão), raramente "eu vou fazer"
- Antecipa objeções antes de serem levantadas
- Valida antes de apresentar solução

### Frases modelo (literais das reuniões)

**Posicionamento:**
- *"Eu não tô olhando aqui só pra gerar demanda pra assessoria. Eu tô olhando a longo prazo pra te transformar em uma referência."*
- *"A nossa assessoria é mais barata do que contratar um designer, um gestor de tráfego, um editor de vídeo e um webdesigner."*
- *"Eu tenho uma assessoria de marketing voltada para profissionais da estética."*

**Processo / educação:**
- *"O investimento em tráfego é um investimento, não um custo."*
- *"eu gosto de ir automatizando a partir do momento que as pessoas estão executando bem o processo"*
- *"É como se fosse as etiquetas do WhatsApp, só que dando pra fazer de uma forma mais geral"* (analogias simples para explicar tecnologia)

**Segurança / tranquilidade:**
- *"A gente não para nada. Continua do jeito que está."*
- *"Vamos embora. Vai dar certo isso daí."*
- *"Vamos fazer um treinamento semanal para que todos fiquem afiados com o CRM"*

**Cadência/ritmo:**
- Frases curtas. Uma ideia por vez.
- Confirma entendimento: *"Vocês rodam campanha para São Paulo, certo? Certo. Show."*
- Responde perguntas sem enrolação: vai direto ao como funciona

### Padrões estruturais de Eric nas reuniões
1. **Ouve primeiro** — deixa o cliente falar sobre o problema
2. **Valida** — "Faz sentido. Eu entendo."
3. **Reframe** — transforma o problema em oportunidade de sistema
4. **Ancora** — usa comparação de custo/valor concreta
5. **Fecha com ação** — próximo passo específico, nunca vago

## 5. INSIGHTS PARA O FUNIL HOF (para @copy-chef)

### Ângulos validados pelas reuniões reais:

**TOFU — Hooks que espelham a dor real:**
- Não usar "visibilidade" ou "posicionamento" — usar a linguagem deles:
  - ✅ "Você passa o dia respondendo WhatsApp e ainda perde lead"
  - ✅ "Você é bom no procedimento. Mas continua dependendo de indicação."
  - ✅ "Você quer produzir conteúdo mas trava na hora de postar"
  - ❌ Evitar: "aumente sua visibilidade", "posicione sua marca"

**MOFU — O argumento que derruba a objeção de preço (validado em múltiplas reuniões):**
- *"Nossa assessoria é mais barata do que contratar designer + gestor de tráfego + editor de vídeo + webdesigner separados"*
- Este argumento foi usado espontaneamente por Eric e funcionou. Usar no funil.

**BOFU — O que faz o cliente decidir:**
- Entender que está pagando mais caro por NÃO ter o sistema (custo de oportunidade)
- Parcelamento / baixa barreira de entrada (R$300 de sinal funcionou para Dra. Gabrielle)
- Segurança: "a gente não para o que já está funcionando"

### Temas de conteúdo validados (aparecem nas próprias reuniões):
1. "Como sair do ciclo de indicação" (dor mais frequente)
2. "Por que lead no WhatsApp sem CRM é dinheiro perdido"
3. "O que muda quando você para de boostar post e começa a ter estratégia"
4. "Quanto custa NÃO ter uma equipe de marketing" (argumento que Eric usa para vender)
5. "Como profissional de estética pode produzir conteúdo sem perder tempo com paciente"
