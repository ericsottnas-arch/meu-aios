## DNA de comunicacao do MarriShow (para copy futura)

**Expressoes caracteristicas:**
- "Se liga ai" — chamar atencao
- "Xama que e sucesso" / "Chama que e sucesso" — CTA
- "Trem" — mineirismo para qualquer coisa
- "Pai" / "Meu filho" — tratamento informal
- "Como e que resiste" — impressionado com algo bom
- "Nao tem preco nao, pai" — expressar valor
- "Vai ficar fino" — algo vai ser muito bom
- "Mocada" — forma de chamar a audiencia
- "Bora" — convite para acao

**Estrutura tipica dos videos:**
1. Abertura impactante sem apresentacao formal
2. Brincadeiras com amigos (apelidos: Boio, Bolota, Boca de Balde)
3. Experimentacao genuina ao vivo
4. Informacao pratica ("vou deixar na legenda")
5. CTA: "Chama que e sucesso"

**Nunca faz:**
- Linguagem formal ou tecnica
- Menciona preco diretamente
- Parece estar lendo roteiro
- Transicao brusca para modo publicitario

## Como foi produzido

1. API mobile do Instagram (`/api/v1/users/web_profile_info/`) para listar posts
2. yt-dlp para baixar audio de 5 videos como MP3
3. Groq Whisper (whisper-large-v3) para transcricao
4. Analise manual das transcricoes para extrair DNA de comunicacao
5. Roteiro escrito com base nas expressoes e estrutura reais dele
